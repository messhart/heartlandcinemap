Heartland Cinemap

Arthouse & repertory showtimes across the Midwest. Tickets are sold by each cinema's own box office — we just point the way.

About

Heartland Cinemap is a free, non-commercial listing of what's screening at independent, arthouse, repertory, and nonprofit cinemas around the Midwest. It exists to send people to these cinemas — every listing links to the venue's own page, which is always the source of truth. We never sell or hold tickets, and showtimes can change: please confirm with the cinema.

Run a cinema? If you'd like your listings corrected or removed, email dyfttym@protonmail.ch and it will be done, no questions asked.

Source code and data on GitHub. ZIP locations from the US Census Bureau (public domain). Film descriptions and metadata from TMDB — this site uses the TMDB API but is not endorsed or certified by TMDB.
